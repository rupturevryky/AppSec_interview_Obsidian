
**База:**
1. Поднимаю gitlab community версию;
2. В CI/CD помещаю Trivy, Gitleaks, Trufflehog и прочие open source сканеры;
3. С внешнки закрываем всё, кроме пользовательских сайтов. Будем ходить через VPN. 
   > Если с тобой сражаются против VPN: используем OAuth Proxy. Настраивается через key cloack, подсовываешь в AD.
4. На все входы 2FA;
5. Убираем все SSH. Администрируют только администраторы внутри сети.

**Продвинутый уровень:**
1. Накатываем open source SIEM, SIEM DB;
2. Поднимаем **HashiCorp Vault** для секретов. Ничего не храним в git-е;
3. WAF, но его ставить муторно;
4. **Fleet** (платформа для централизованного управления и мониторинга состояния тысяч устройств) каждому на тачку;
