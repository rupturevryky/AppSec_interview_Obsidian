>**Вопрос:**
 >
 >Можно ли имея C2 сессию терминального сервера под локальным админом доменного сервера доменной учётной записи дойти до доменного администратора без логинов и паролей.
 
 ---
**Ответ:**

**Список процессов, запущенных на терминальном сервере.**

> Каждый процесс запущен из под какого-то пользователя. Как только ты делаешь инжект в процесс - ты own-нишь токен владельца этого процесса. Соответственно на этом сервисе ты можешь получить права windows server admin -> переместиться на другой сервер -> дойти до доменного админа.
